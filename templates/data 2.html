Data Instructions for Atlantis!

For your data section, upload a CSV with the following columns:
  • All the sample names (Any column that is not labelled 'gene', 'log2f' or 'p_values' will be taken as a Sample.).
  • 'gene' which denotes the gene name.

    • Optional fields:
  • log2f which denotes the log fold change.
  • p_values which denotes the p_values for the differential expression of that gene.

    • Data Annotation Instructions for Atlantis!

    For your data section, upload a CSV with the two columns:
  • Sample Name: This column should contain all the corresponding sample names.
  • Condition: This column should contain the numerical indication of the condition that the sample was placed in. Eg. 1, 2, 3 etc.
